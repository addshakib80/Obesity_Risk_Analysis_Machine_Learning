# Obesity Risk Analysis Using Machine Learning

## Overview
This project analyzes an obesity-related dataset to study the relationship between lifestyle, behavioral, and physical factors and obesity levels.

## Objectives
- Explore obesity and weight-related patterns in the data
- Perform data preprocessing and exploratory data analysis (EDA)
- Train and evaluate machine learning models for obesity classification/prediction

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Methodology
- Data cleaning and encoding
- Feature engineering and EDA
- Supervised machine learning model training
- Performance evaluation using quantitative metrics

## Domain
Health Analytics / Medical Data Science
